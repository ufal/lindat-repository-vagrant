Few notes and facts

  • Warning
    Use proper hostnames e.g., dspace.lindat.dev to access the repository otherwise few things will not work properly (including some links and autocomplete)!
  • credentials 
    # nagios               - nagiosadmin          | dspace
# postgresql           - dspace               | dspace
# tomcat/nagios/probe  - tomcat               | dspace
# monit                - admin                | dspace
# dspace               - dspace@lindat.cz     | dspace
# piwik                - lindat               | lindat
  • hostname
    lindat5 box: dspace.lindat5.dev
    lindat box: dspace.lindat.dev
    dspace4 box: dspace.dev
  • ip address (33.33.33.XX)
    lindat5 box: 33.33.33.80
     lindat box: 33.33.33.78
    dspace4 box: 33.33.33.79
  • The host for the lindat5 box is dspace.lindat5.dev so update your hosts file (e.g., on windows add
    33.33.33.80 dspace.lindat5.dev
    to c:\Windows\System32\drivers\etc\hosts). The host for the dspace4 box is dspace.dev
  • solr was made available by removing filter LocalHostRestrictionFilter from tomcat's webapps/solr/WEB-INF/web.xml - in case of redeploying you need to do it again
